With Flex, you'll discover that far less CSS is required to achieve certain layouts

The Flex Container behaves like a block element for the most part. It'll take up the entire width available by default. Its children, Flex Items have a special behaviour. Think of the Flex Container as the choreographer and Flex items as the dancers.
